Tentang


Gelanggang Inovasi dan Kreativitas, sering disebut sebagai Gedung Baru Gelanggang, merupakan salah satu fasilitas terbaru di Universitas Gadjah Mada (UGM) yang berfungsi sebagai pusat kegiatan inovasi dan kreativitas mahasiswa. Gedung ini didesain untuk mendukung berbagai aktivitas akademik, seni, budaya, dan kewirausahaan di lingkungan kampus. Terletak di kawasan kampus UGM, Yogyakarta, bangunan ini memiliki desain arsitektur modern dengan luas bangunan yang cukup besar serta struktur yang kompleks.

Mengapa Gelanggang Inovasi dan Kreativitas?

Sebagai pusat kegiatan inovasi, seni, budaya, dan kewirausahaan, Gedung Gelanggang memiliki nilai strategis untuk didokumentasikan dalam bentuk model 3D. Model ini dapat digunakan sebagai referensi untuk perencanaan tata ruang kampus, manajemen aset, dan pengembangan infrastruktur.

Membuat 3D Model LoD-2

Menghasilkan model 3D bangunan dengan detail hingga elemen atap dan dinding, menggunakan data point cloud dari LiDAR untuk memperoleh representasi visual yang akurat.

Membuat Ground Surface

Mengolah data LiDAR untuk membangun permukaan tanah (ground surface) yang bebas dari objek permukaan, dengan akurasi tinggi guna mendukung konteks spasial model.

Mengintegrasikan Model 3D dan Ground Surface

Menggabungkan model 3D bangunan LoD-2 dengan data permukaan tanah untuk menciptakan representasi spasial terpadu yang dapat digunakan dalam berbagai analisis spasial dan aplikasi geoinformatika.

Tahapan Pengerjaan

Diagram Alir




1. Akuisisi data menggunakan LiDAR UAV Zenmuse L2 dan menghasilkan data point cloud
2. Post-processing data menggunakan perangkat lunak DJI Terra
3. Pemisahan point cloud antara objek ground dan non-ground menggunakan perangkat lunak Cloud Compare dengan algoritma CSF Filter
4. Pembuatan DTM dari point cloud ground menggunakan perangkat lunak Global Mapper dengan algoritma Blue Marble Geographics Hybrid Filtering (BMHF)
5. Pembuatan 3D-building GIK dengan menggunakan perangkat lunak SketchUp
6. Pengintegrasian dan visualisasi antara ground surface DTM dengan 3D-building GIK menggunakan CesiumJS
